Refunds∵

Issue a Refund∵

Issues a refund from a debit. You can either refund the full amount of the debit or you can issue a partial refund, where the amount is less than the charged amount.

amount
optional integer. USD cents. Defaults to null.
description
optional string.
meta
optional key-value. Single level mapping from string keys to string values. Defaults to {}.
debit_uri
optional string. USD cents. If the resolving URI references a debit then that is used as the default. Otherwise no default is provided and this field is required.

Definition

POST https://api.balancedpayments.com/customers/:customer_id/refunds

balanced.Debit.refund()
